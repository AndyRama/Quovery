How does it work?

Qovery Engine is an Open-source abstraction layer product that turns easy apps deployement on AWS, GCP, Azure and others Cloud providers.The Qovery Engine is coded in Rust and takes advantage of terraform, Helm , Kubernet, Docker to manage resources.

Usage

Initiaize EKS ( AWS Kubernet) and ECR (AWS container registry) on AWS.

Deploy real world application

Qovery Engine empowers you to deploy complex applications, such as a backend, a frontend, and a database in a very simple way.

Servers, networking, security, all done by Engine for you

Resiliency is key

Qovery Engine knows when something goes wrong on the deployement of your applications. Qovery Engine is built withresiliency in mind.

A transaction Engine inspired by what is provided into ACID databases has been developed at the heart of the product to rollback on a consistent and working application version when something goes wrong.
